# PRODIGY_ML_01: House Price Prediction Model

This repository contains **Task 01** completed during my Machine Learning Internship at **Prodigy InfoTech**.

## 📋 Task Overview
Implement a linear regression model to predict house prices based on three fundamental features:
* **Square Footage** (`GrLivArea`)
* **Bedrooms** (`BedroomAbvGr`)
* **Bathrooms** (Engineered from `FullBath` + `HalfBath`)

## 📊 Dataset
The dataset utilized is the popular [House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques) dataset from Kaggle.

## 🛠️ Performance Metrics
* **R-squared ($R^2$) Score:** 0.6286 (~63% of variance explained by these three features alone)
* **Root Mean Squared Error (RMSE):** $53,371.56
